자바스크립트 선언문

선언문은 자바스크립트 코드를 작성할 영역을 선언하는 것이라고 이해하면 됩니다.<script>라는 태그로 선언문이 시작된 곳부터</script>라는 태그로 종료된 곳까지를 스크립트 영역이라고 부릅니다.

<script>
document.write("환영합니다");
</script>

자바스크립트 주석처리

자바스크립트 선언문 안에서 코드가 아닌 설명글을 넣고 싶을 때는 주석 처리를 하면 됩니다. 이때 한 줄 주석일 경우에는 //한 줄 설명글로 작성하고, 여러 줄 주석일 경우에는 /*여러 줄 설명글*/로 작성하면 됩니다.

//한 줄 설명글인 경우
/*
설명글이 여러 줄인 경우
이렇게 처리합니다.
*/
<!--HTML 소스의 설명글은 이렇게 처리합니다-->

외부 스크립트를 내부로 가져오기

외부 스크립트를 안으로 불러와서 작업 할 때는 해당 스크립트의 경로를 지정하여 코드를 작성해야 합니다.

<script src="JS 파일 경로"></script>

코드 입력시 주의사항

1. 자바스크립트는 대.소문자를 구분하여 작성합니다.

날짜 객체 생성 : New date( ); (X)
날짜 객체 생성 : new Data( ); (O)

2. 코드를 작성한 후에는 세미콜론 ( ; )을 쓰는것이 좋습니다.
코드를 작성할 때에는 한 줄에 한 문장만 작성하는게 좋습니다.

documnet.write("hi")     (X)
document.write("hi"); documnet.write("bye"); (X)

document.write("hi");    (O)
documnet.write("bye"); (O)

3.문자형(String) 데이터를 쓸 때는 큰따옴표와 작은따옴표를 구분해야합니다.

큰따옴표 겹침 오류
documnet.write("책에 "자바스크립트는 대소문자를 구분해야 합니다"라고 나와 있다.");
좋은 예
document.write('책에 "자바스크립트는 대소문자를 구분해야 합니다"라고 나와 있다.');

4. 코드를 작성할 때는 괄호의 짝이 맞아야 합니다.

document.write("welcome!";   (X)
document.write("welcome!");  (O)
