속성 설명
grid grid의 속성들을 한번에 일괄적용 할 수 있는 속성입니다.
grid-area grid의 요소들을 배치할 수 있는 속성으로 row/column 또는 grid-template-areas에 등록된 이름을 사용할 수 있습니다.
grid-auto-columns 설정하지 않은 그리드의 행/열 트랙 크기를 자동으로 설정합니다.
grid-auto-flow 설정하지 않은 레이아웃이 있을 경우, 자동 배치 알고리즘이 실행되어 자동으로 배치 되도록 설정할 수 있습니다.
grid-auto-rows 설정하지 않은 그리드의 행/열 트랙 크기를 자동으로 설정합니다.
grid-column 공백으로 구분된 값 리스트를 해석하여 그리드 열(Column)을 설정합니다.
grid-column-end 그리드 칼럼라인을 기준으로 해서 배치할 영역을 설정합니다. start 부터 end 까지의 지점을 설정합니다.
grid-column-gap 그리드 라인 칼럼의(행/열) 사이 간격(gap)을 조정합니다.
grid-column-start 그리드 칼럼라인을 기준으로 해서 배치할 영역을 설정합니다. start 부터 end 까지의 지점을 설정합니다.
grid-gap 그리드 라인(행/열) 사이 간격(gap)을 조정합니다.
grid-row 공백으로 구분된 값 리스트를 해석하여 그리드 행(Row)을 설정합니다.
grid-row-end 그리드 로우 라인을 기준으로 해서 배치할 영역을 설정합니다. start 부터 end 까지의 지점을 설정합니다.
grid-row-gap 그리드 라인 로우의(행/열) 사이 간격(gap)을 조정합니다.
grid-row-start 그리드 로우 라인을 기준으로 해서 배치할 영역을 설정합니다. start 부터 end 까지의 지점을 설정합니다.
grid-template 그리드 템플릿 행 / 열 / 영역 이름 설정을 속기형으로 설정 가능합니다.
grid-template-areas 설정된 그리드 영역의 이름을 참조하여, 그리드 템플릿 영역을 설정할 수 있습니다.
grid-template-columns 공백으로 구분된 값 리스트를 해석하여 그리드 열(Column)을 설정합니다.
grid-template-rows 공백으로 구분된 값 리스트를 해석하여 그리드 행(Row)을 설정합니다.
